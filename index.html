Overview

ExplainBind is an interaction-aware framework for proteinâ€“ligand binding (PLB) prediction. It supervises token-level cross-attention using non-covalent interaction maps derived from curated PDB proteinâ€“ligand complexes in InteractBind. By aligning model attention with physically grounded interaction signals, ExplainBind transforms PLB prediction from a black-box reasoning into a chemistry-grounded process suitable for large-scale screening.
